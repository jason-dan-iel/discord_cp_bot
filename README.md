# discord_cp_bot

This is a work in progress bot for programming club

## Format for this bot
1. Clone this repo
2. clone dpp in libs/ directory "git clone https://github.com/brainboxdotcc/DPP.git"
3. Change CmakeLists.txt and ./src/source.cpp and build
