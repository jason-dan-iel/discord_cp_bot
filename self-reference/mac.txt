    brew install cmake

        link openssl:

final building process
    1. cmake -B ./build
    2. cmake --build ./build -j8
    3. cd build; sudo make install
    4. clang++ -std=c++17 -o bot ./src/source.cpp -ldpp
    5. ./bot
